## 🌟 Features

- **Smart Progress Rings:** Interactive calorie and macronutrient tracking (Protein, Carbs, and Fats) using Apple Watch-inspired concentric progress rings.
- **Meal Management:** Browse daily meals, mark completed meals, and save favorites.
- **Food Database & Search:** Search through the built-in food database and instantly view nutritional information and macros.
- **AI Chat Companion:** An integrated AI assistant that helps customize your diet, answer nutrition questions, and suggest food substitutions.
- **Calendar & Food Swap Rules:** Easily navigate your meal plan by day and access predefined food replacement options.
- **Dark & Light Mode:** Smooth theme switching for a comfortable viewing experience.
- **Fully Responsive Design:** Optimized for desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** Vanilla CSS3 (Custom Properties, Flexbox, Grid, Animations)
- **Logic:** Vanilla JavaScript (ES6)
- **Typography:** Google Fonts (Tajawal & IBM Plex Sans Arabic)
- **Icons:** SVG Vector Graphics
